# Flink CSV to JDBC Table API Demo
Demonstration of Flink's Table API for CSV to JDBC operations<br>
Data sourced from the [Zillow ZEstimate Kaggle Competition](https://www.kaggle.com/c/zillow-prize-1/data)

## Usage
```bash
mvn spring-boot:run
```
